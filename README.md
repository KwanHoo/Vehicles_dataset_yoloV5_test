# Vehicles-OpenImages_dataset_yoloV5_test


- YOLO object detection test using Vehicles dataset
- dataset contains 627 images of various vehicle classes for object detection
- train images 갯수 : 878
- test images 갯수 : 250
- 모델 : yolo v5
- image size : 640
- batch :16
- epochs : 40
